## â• Adding Elements <br>
**append()** <br>
Adds element at the end.

**insert()** <br>
Adds element at a specific index.

## â– Removing Elements
**remove()** <br>
Removes first occurrence of value.
```python

```
**pop()** <br>
Removes element by index (default: last).

**clear()** <br>
Removes all elements.
```python

```

**del** <br>

